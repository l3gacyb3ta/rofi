## Usage

1. Clone this repository locally
2. Copy `catppuccin.rasi` to `~/.config/rofi/`
3. Run rofi with `-theme catppuccin`
4. Or, add `@theme "catppuccin` to your `~/.config/rofi/config.rofi`
